  • connectShouldFetchUserData(withToken:)

    This method is fired off when the ConnectSession receives the JWT token from the server. At this point we ask the delegate whether we should continue to fetch the user data using the JWT token. Returning true will allow the ConnectSession to retrieve the user data, while false ends the session.

    Declaration

    Swift

    func connectShouldFetchUserData(withToken token: String) -> Bool

    Parameters

    token

    The JWT token returned from the server.

    Return Value

    True allows the session to retrieve the user data, otherwise false ends the session.
